# Smart Parking System Monorepo

This repository is the monorepo for the Smart Parking System (COE892 project).

## Structure
- `architecture.md` — overall system architecture
- `COE892 - Project Proposal.*` — project proposal docs
- `db_rl_proposal.txt` — RL research notes
- `pricing-service/` — pricing microservice (see pricing-service/README.md)

## Adding Components
Add new services (e.g., inventory-service, gateway) as subfolders. Each service should have its own README and docs.

## Quickstart
See pricing-service/README.md for pricing service setup and usage.
